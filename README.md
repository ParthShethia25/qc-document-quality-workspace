# Document Quality Control Workspace (V1)

This project demonstrates an end-to-end **Product Management & Product Analytics** solution for handling failed or suspicious business documents in an AI-powered document processing platform.

The work was created as part of a **Product Manager take-home assignment** and showcases PRD thinking, UX design, API contracts, metrics, rollout planning, and a functional prototype.
